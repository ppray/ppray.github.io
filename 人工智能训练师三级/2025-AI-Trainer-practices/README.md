# 2025年上海交大人工智能训练师（三级）题库

本项目包含2025年上海交通大学人工智能训练师（三级）操作技能模拟试题的完整题库，提供详细的Jupyter Notebook答题过程和模拟数据集，适用于备考练习和教学参考。

## 项目结构
```
practices/
├── 1.1.1_智能医疗系统中的业务数据处理流程设计  # 医疗数据分析
├── 2.1.1_智慧交通中燃油效率模型的数据清洗和标注流程设计  # 交通数据清洗
├── 2.2.1_智能信用评分Logistic回归模型开发与测试  # 信用评分模型
├── 3.1.1_智能音箱产品的数据分析与优化  # 智能音箱数据分析
├── 3.2.1_图像识别评估系统交互流程设计  # 图像识别系统
└── 4.2.1_智能零售分析系统数据采集和处理指导  # 零售数据分析
```

## 使用说明
1. 安装依赖：`pip install -r requirements.txt`
2. 打开各目录下的`example.ipynb`文件查看完整解题过程
3. 每个练习都包含模拟数据集和完整的代码实现

## 特点
- 完整覆盖6大实操题型
- 包含真实模拟数据集（20+行）
